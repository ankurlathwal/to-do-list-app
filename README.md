# To-do List App
## Built using Node.js (Express REST API), ReactJS, Bootstrap and MongoDB
### Platform - Heroku 


Link to App
---

```
https://uncorkd-project.herokuapp.com/
```


Setup
---
 
```
npm install
```
 
 
Compile
---
 
```
npm run compile
```

Usage
---
 
Start the server
 
```
npm start
```
