#To-do List App
##Project for Uncorkd
###Built in Node.js (Express REST API), ReactJS, Bootstrap and MongoDB
####Platform - Heroku 
####Codebase - BitBucket

Link to App
---

```
https://uncorkd-project.herokuapp.com/
```


Setup
---
 
```
npm install
```
 
 
Compile
---
 
```
npm run compile
```

Usage
---
 
Start the server
 
```
npm start
```
